## 🌍 Where the Karoo Breathes

A modern, responsive tourism web application showcasing the beauty and experiences of **Prieska**, located in the **Northern Cape, South Africa**. This platform allows visitors to explore unique travel experiences, book activities, and discover the ancient land of vast skies, amber Kalahari sunsets, and untouched wilderness.

🔗 **Live Demo:** [View Demo](#) *https://prieska-tourism.netlify.app/*

---

## ✨ Features

### 🎯 Core Functionality
- **Interactive Booking Widget** - Date picker, guest selection, and experience filtering
- **Dynamic Experience Filtering** - Browse by category (Wildlife, Adventure, Culture, Stargazing, Heritage, Nature)
- **Real-time Feedback** - Toast notifications for booking actions
- **Smooth Scrolling Navigation** - Seamless page navigation with anchor links

### 🎨 UI/UX Highlights
- **Sticky Navbar** - Dynamic styling that changes on scroll
- **Animated Stats Counter** - Scroll-triggered number animations
- **Responsive Gallery** - Grid layout with hover zoom effects
- **Testimonial Section** - Guest reviews with star ratings
- **FAQ Accordion** - Collapsible Q&A section
- **Newsletter Signup** - Email subscription form

### 📱 Fully Responsive
- Mobile-first design approach
- Optimized for all screen sizes (mobile, tablet, desktop)
- Touch-friendly interface

## 📦 Installation & Setup

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Clone & Install

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/prieska-tourism.git

# Navigate to project directory
cd prieska-tourism

# Install dependencies
npm install

# Start development server
npm run dev
